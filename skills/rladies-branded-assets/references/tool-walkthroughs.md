# RLadies+ asset tools: step-by-step

Deep reference for `rladies-branded-assets`. Sources: `branding/canva-logo`,
`affinity-logo`, `hex-logos`, `canva-social-media`, `slidedeck-templates`,
`chapter-branding`. Logo-integrity and accessibility rules from `rladies-brand` apply
to all of these.

## Customising the R+ logo

**Canva** (`branding/canva-logo`) — fill the R and + of the submark with images or
brand colours. Free Canva cannot export transparent backgrounds (use Affinity if you
need transparency). Insert images into the R/+ shapes, recolour from the brand kit,
then **Share → Download → PNG/JPG** (GIF for animation). Rule from the guide: _"Please
do not move, transform, or rotate the R relative to the RLadies+ text part of the
logo."_

**Affinity Studio** (`branding/affinity-logo`) — free, and exports transparent PNG.
Edit the **Content** layer (the Purple outline layer is locked); use clipping masks to
place images inside the R/+; recolour with the brand hex codes; export as PNG.

## Chapter hex stickers

`branding/hex-logos` — download the Affinity hex template, edit the **Content** layer
with the chapter name and any decorative elements, recolour using the brand hex codes
(batch-select strokes via **Layer ▸ Select ▸ Same ▸ Stroke Colour**), export as PNG.
Alternative: the Canva R+ template, inserting chapter photos into the R and + shapes.
There's also a Shiny hex maker (`connect.thinkr.fr/hexmake`).

## Social-media graphics

`branding/canva-social-media` and the
[RLadies+ Canva workspace](https://www.canva.com/folder/FAHCPq0DH3w) — use the published
**Brand Templates**: Brand ▸ Brand templates ▸ _Use this template_ (makes your own copy)
→ pick the page sized for your platform → edit text/images → **Share → Download →
PNG/JPG** (GIF for animated). The full template list and per-platform sizes are in
`rladies-social-posts` (`references/platforms-and-buffer.md`).

## Slide decks

`branding/slidedeck-templates` — copy the Google Slides template (browser, free) or
download the PowerPoint template (Poppins embedded). Each layout has three colour
variations (light / purple / dark). Fill the heading/content placeholders, insert
images with the crop tool and logos (institutional logos may sit alongside the R hex),
add slide numbers, optionally animate for video, then export PPTX/PDF (or PNG/JPG/GIF
for image posts). No italics; bold only; high contrast.

## Documents & certificates

`templates/documents` (with and without letterheads) and `templates/certificates`
(DOCX and PDF) in the branding-materials repo — same rule: edit placeholders only.
